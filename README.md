# xs2a-react-native

Integrate FinTecSystems' XS2A into your React Native App.

## Installation

```sh
npm install xs2a-react-native
```

## Usage

```js
import Xs2aReactNative from "xs2a-react-native";

// ...

const result = await Xs2aReactNative.multiply(3, 7);
```
